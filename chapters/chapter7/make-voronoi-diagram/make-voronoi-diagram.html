7.8: Funktion zur Konstruktion des Voronoi--Diagramms für eine Punktwolke

Funktion zur Konstruktion des Voronoi--Diagramms für eine Punktwolke: Die Punkte werden zuerst lexikographisch (von oben nach unten, dann von rechts nach links) sortiert. Durch Abarbeiten dieser Punkte in einer for-Schleife wird also automatisch das Herabsinken der Leitlinie für die site-events richtig umgesetzt; aber natürlich müssen zwischendurch allfällige circle-events behandelt werden. Das Voronoi-Diagramm selbst wird als dict konstruiert, dessen keys die Paare von (Indices von) Punkten sind, deren Streckensymmetrale eine Kante $e$ des Voronoi-Diagramms enthält, und dessen values Punkte (also Knoten) des Voronoi-Diagramms sind, die diese Kante $e$ begrenzen (zumindest auf einer Seite). In den Kommentarzeilen ist die Vorgangsweise genau beschrieben, darum ist der Code auch so lang.
